# Habit App API

A Habit App API that acts as the back-end for a full stack React Native app using [MongoDB atlas](https://www.mongodb.com/atlas/database) for our database.

## Link to hosted version

https://habit-app-api-gg3y.onrender.com/api

## Installation

1. Make sure that [Node.js](https://nodejs.org/en) is installed:

2. Fork and clone this repo
3. Install the dependencies using:

```
npm install
```

The minimum version of Node.js is:

```
v20.3.1
```

## Running tests

Run tests using:

```
npm test
```

## Create an .env files in Project Root directory

- In order to access the databases we need to create an .env files:

Create a file called .env that contains the following:

```
DATABASE_URL=<MongoDB Connection string>
```
